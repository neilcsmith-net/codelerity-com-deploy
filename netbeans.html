Apache NetBeans packages provided by Neil C Smith of codelerity. Neil is a member of the Apache NetBeans PMC and of the Apache Software Foundation. These packages are built with NBPackage using the official Apache NetBeans binary zip release. The packages themselves are not released by the foundation.

These installers include a local Java {{ page.jdk-version }} JDK for the IDE to run on, offering a self-contained out-of-the-box experience. Additional JDK's for projects may be registered and/or downloaded within the IDE, supporting development of projects from Java 8 up.

Support

For issues with the installers / packages themselves, please raise an issue on our GitHub repository.

For community support with the IDE itself, please use the Apache NetBeans mailing lists.

For commercial support, maintenance or development with the IDE or platform, please contact us.
We can also provide customized and supported installers with a JDK and cluster configuration of your choice.

Build your own

All our installers are built using the GitHub workflow at https://github.com/codelerity/netbeans-packages. This is also a template. Feel free to fork, edit and run this workflow to customize and build your own installer packages. Although if you distribute them, please make sure to remove references to Codelerity from the configuration!
